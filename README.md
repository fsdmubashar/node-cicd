## 📋 Overview

Yeh ek simple Node.js application hai jo **CI/CD pipeline testing** ke liye banai gayi hai. Is repository mein:

- ✅ Basic Node.js application
- ✅ GitHub Actions workflow
- ✅ Automated testing
- ✅ Build automation
- ✅ Deployment pipeline

---

## 🚀 Quick Start

```bash
# Clone repository
git clone https://github.com/fsdmubashar/node-cicd.git
cd node-cicd

# Install dependencies
npm install

# Run application
npm start

# Run tests
npm test
```

---

## 📦 Prerequisites

```bash
Node.js 18+
npm 9+
Git
```

---

## ⚙️ Commands

### Development

```bash
# Install dependencies
npm install

# Start server
npm start

# Start with auto-reload
npm run dev

# Run tests
npm test

# Run tests with coverage
npm run test:coverage

# Lint code
npm run lint
```

---

## 🔄 CI/CD Pipeline

### GitHub Actions Workflow

Yeh repository GitHub Actions use karta hai for:

1. **Automated Testing** - Har commit par tests run hote hain
2. **Code Linting** - Code quality check
3. **Build Process** - Application build
4. **Deployment** - Auto deployment (optional)

### Workflow Triggers

```yaml
- Push to main/master branch
- Pull requests
- Manual trigger
```
